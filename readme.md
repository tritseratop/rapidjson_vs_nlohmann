# rapidjson_vs_nlohmann

## Приложение

Сравнение быстродействия парсинга в json библиотеками rapidjson и nlohmann

## Описание кода

- Преобразуемые структуры и преобразовывающие функции описаны в файлах **values_description**
- Тесты описаны в **tests.h**
- В файлах **utility** описаны функции, создающие векторы различных типов
- В файле **constants.h** объявлены константы используемые в тестах

## Описание тестов

1. Тестировалось преобразование в json видов:
	- { "tws": 1945, "d": [ {"v": 1}, {"v": 2}, {"v": 3}, {"v": 4}, {"v": 5} ] } (как в AtechSCADA)
	- { "tws": 1945, "d": {"v": [1, 2, 3, 4, 5, 6]} }
2. Результаты:
	- Для формата json принятого в AtechSCADA **nlohmann быстрее**, чем rapidjson 
	- Для формата json во втором варианте **rapidjson быстрее**, чем nlohmann
